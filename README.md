# System of INN management - JDBC 
Project developed to practice: - Java interlinked with Mysql - JDBC (Java Database Connectivity) 

## Features:
* Register of Clients, Apartments and Reservations 
* Table reading
* Table Updates
* Canceling of reservations

## Tecnologies
Java 25 + JDBC + Mysql 

## Technical Details 
* JDBC
* Connection
* PreparedStatement
* ResultSet
* SQLException
* IOException

## Execution
Open the project in NetBeans and execute: Main.java
